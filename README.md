# Java Projects Collection
My Java projects through out my high school years.

## Project Categories

### UIL Practice and Competition
Programs written while preparing for or competing in UIL Computer Science competitions. These include logic exercises, algorithm implementations, randomization experiments, and other small programs.

### Personal Projects
Independent programs created out of curiosity or experimentation. These range from small utilities to exploratory code meant to test ideas or learn new concepts.

### School Related
Java exercises completed for classwork or structured practice, and other school related codes, focusing on basic fundamentals such as loops, conditionals, arrays, and object‑oriented programming.

## Purpose

This repository serves as a portfolio of my growth as a programmer. Many of these programs are small, focused exercises, but together they represent consistent practice, self‑teaching, and preparation for competitive programming. Most of the codes included here were created out of my pure passion for computer science.

## Technologies Used
- Java (JDK 8+)
- Eclipse IDE
- Basic algorithms and data structures
- Random, input, output, loops, conditionals, arrays

## Notes

These projects are intentionally simple and lightweight. They reflect my progression as a student learning Java through hands‑on experimentation, UIL CS preparation, and personal interest in computer science.
